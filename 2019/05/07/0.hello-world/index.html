Hello World

2019-05-07 扩展阅读 扩展阅读 Comments

前言

  • 入了前端坑了以后，才越发发现技术领域的庞大，为了巩固所学知识 与项目实战的经验 着手准备撰写博客

  • 该博客采用Hexo框架搭建，配置在了全球最大同性交友平台（giyhub）上

搭建流程概述

  1. 安装nodejs 官网
  2. 打开cmd，安装hexo 官网
    • npm install hexo-cli -g
  3. 新建文件夹 在cmd中初始化hexo
    • hexo init
  4. 安装对应主题（去github或官网查看）
  5. 配置_config.yml
  6. 使用git 初始化、上传到github上
